# wpAtom - WIP
## Simple Theme Starter, that doesn't get in your way.

###Packed with a lot of goodies:
- webpack + babel-loader
- sass
- bootstrap sass
- composer
- no bower

###Start:
- npm install
- gulp dev
- webpack

### Dist
- gulp dist --themeName themeName
