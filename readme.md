# wpAtom - 0.0.1
A wordpress start theme.
Designed to provide minimal boilerplate, to use the newest front-end technologies.

###Packed with a lot of goodies:
- webpack
- es6
- sass
- bootstrap sass
- composer

###Start:
- npm install
- gulp dev
- webpack
